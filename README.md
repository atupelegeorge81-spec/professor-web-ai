# Professor AI - Web Based

🤖 **Professor** is your confident, no-nonsense AI co-pilot built with Groq API.

## Features

✨ **Futuristic UI**
- Dark mode with neon accents
- Smooth animations and transitions
- ChatGPT-style message display
- Fully responsive design

🧠 **Intelligent Assistant**
- Powered by Groq's Mixtral-8x7b model
- System prompt with "Binadamu-AI" personality
- Confident, direct, opinionated responses
- No robotic filler phrases

💾 **Local Storage**
- Conversation history saved to browser localStorage
- Up to 20 recent conversations
- Auto-save after each message

🚀 **Ready to Deploy**
- GitHub Pages compatible
- No backend required
- Static HTML, CSS, JavaScript

## Quick Start

1. Clone this repo
2. Open `index.html` in your browser
3. Start chatting with Professor!

## Configuration

Edit `config.js` to:
- Change the Groq API key
- Adjust model parameters (temperature, max tokens)
- Customize the system prompt

## System Prompt Philosophy

Professor operates with these core principles:

1. **No Robotic Phrases** - Jumps straight to the answer
2. **Has Opinions** - Challenges bad ideas constructively
3. **Matches Energy** - Adapts to your tone and style
4. **Admits Mistakes** - No over-apologizing
5. **Thinks Out Loud** - Shows reasoning process

## Deployment

### GitHub Pages

1. Go to repo settings → Pages
2. Set source to `main` branch
3. Your app will be live at `https://yourusername.github.io/professor-web-ai`

## Tech Stack

- **Frontend**: HTML5, CSS3, Vanilla JavaScript
- **API**: Groq (OpenAI-compatible)
- **Storage**: Browser LocalStorage
- **Hosting**: GitHub Pages
